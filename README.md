# Лабораторная работа №9

Приложение представляет удобный и интерактивный список игр с описанием и названием.


## В прилоежеини использовались

- Column
- row
- card
- image
- text
- button
- spacer
- materialTheme

# Логика
- приложение хранит состояние системы с помощью переменной и by remember { mutableStateOf(0) }
- Кнопки переключения вперед назад которые меняют индекс состояния и пересоздают приложение по новому состоянию
